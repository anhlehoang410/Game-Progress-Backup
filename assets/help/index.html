Before we begin

Not all game progresses can be saved without having root privileges, so if you want to backup everything you have to root your device. Game support will grow, be patient!

Why backup?

I can think about two good reasons to do a backup: to not lose your progress and to move your progress to a different device and continue playing.

Why don't root?

Root gives to you complete power on your phone (from which "comes great responsibility" - thank you Stan Lee), but you will lose warranty, so you could think twice rooting it if you are not an advanced user.

Backup

To backup progresses of a supported game you have to choose it from the list and click on it. It will appear a menu from which you'll have to select "Backup to SD CARD". Files will be saved in "/sdcard/game_progress_backup/.

Restore

To restore an existing backup you have to:

1) Run the game and begin playing so that the game will create for you the files you need (you can do it yourself, because you haven't got root permissions)

2) choose the game from the list and click on it. It will appear a menu from which you'll have to select "Restore from SD CARD".

Scripts

Scripts are intended to run a task. To use a script you have to select "Scripts" tab and select it from the list. Clicking on a script will show his description. To run a script, click on it and select "Run" under the description.
